# Country Reporter
A web app that leverages AWS's services to manage country data and create comprehensive reports within a Country and Global Scope

# Tech Stack
 - React.js (Frontend)
 - Python (Backend) 
 - Terraform (IaC)
 - Docker (Container)
 - YAML (CI/CD)
 - AWS (Services used):
   - EKS
   - DynamoDB
   - VPC
   - Lambda
   - S3
   - IAM
